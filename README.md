# ONIRO Colab

Public mirror for running ONIRO URM on Colab.

## Setup
1. Open `oniro_colab_micro.ipynb` in Colab
2. Runtime → Change runtime type → T4 GPU
3. Run All

## What it does
- URM (Universal Reasoning Model) recursive transformer
- RIMA Reweighter, DIS deep supervision, AlphaEvolve-Gödel
- Trained on ARC-AGI-1 + ARC-AGI-2 + Sudoku + Math
- Evaluated on all 4 datasets
